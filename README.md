# dsltogls

Convert files from DSL to GLS format

### Usage
```
dsltogls dsl_file gls_file
```

### How to build
#### Ubuntu
```
git clone https://github.com/lingvoboard/dsltogls.git
cd dsltogls
/usr/lib/x86_64-linux-gnu/qt5/bin/qmake && make
```
#### Arch Linux, Manjaro
```
git clone https://github.com/lingvoboard/dsltogls.git
cd dsltogls
qmake-qt5 && make
```
